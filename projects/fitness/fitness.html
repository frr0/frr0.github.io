Fitness Analytics

This project is dedicated to the analysis of fitness data. It provides tools and notebooks for tracking, visualizing, and understanding workout and health metrics.

Key features

  • Data import from different fitness sources
  • Data visualization for trend analysis
  • Performance analysis across workouts
  • Health metrics over time
